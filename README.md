# 🎮 VoidTrigger

> FPS + 로그라이크 요소가 결합된 액션 서바이벌 게임
> Unreal Engine 기반 C++ 프로젝트

---

## 📌 프로젝트 소개

**VoidTrigger**는 플레이어가 끊임없이 몰려오는 적들을 상대로 생존하며
레벨업을 통해 능력을 강화해 나가는 FPS 로그라이크 게임입니다.

* 수동 조작 시스템
* 랜덤 업그레이드 기반 성장 구조
* 웨이브 기반 적 스폰 시스템
* 보스 전투 포함

---


## 🕹️ 주요 기능

### ⚔️ 전투 시스템

* FPS 기반 직접 조작

### 📈 성장 시스템

* 레벨업 시 랜덤 업그레이드 선택


### 🧠 게임 시스템

* 웨이브 기반 스폰 시스템
* 경험치 및 레벨업 구조
* UI (HUD, 레벨업 선택창)

---

## 🛠️ 사용 기술

* **Unreal Engine 5**
* **C++**
* Blueprint (UI 및 일부 로직)

---

## 🧱 프로젝트 구조

```
VoidTrigger/
├── Content/        # 에셋 및 블루프린트
├── Source/         # C++ 코드
├── Config/         # 설정 파일
└── VoidTrigger.uproject
```

---

## 🚀 실행 방법

1. Unreal Engine (버전: 5.7) 설치
2. 레포 클론

```
git clone https://github.com/minggong222/VoidTrigger.git
```

3. `.uproject` 파일 실행
4. (필요 시) Visual Studio 프로젝트 생성

   * 우클릭 → Generate Visual Studio project files
5. 에디터에서 Play 실행
